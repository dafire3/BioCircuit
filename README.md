# BioCircuit

A concept demo for an archaeology tool that uses slime-mold-inspired AI to suggest promising paths from a known archaeological site.

## Tech Stack

- **Next.js 14+** (App Router) with TypeScript
- **TailwindCSS** for styling
- **Framer Motion** for animations

## Setup

1. Install dependencies:
```bash
npm install
```

2. Place your logo file at `public/biocircuit-logo.png`

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Features

- **Logo Intro Animation**: Pixar-style bounce animation with ground crack effect
- **File Upload**: Drag-and-drop or click to select maps/LiDAR scans
- **5-Question Wizard**: Smooth animated transitions between questions
- **Result Screen**: Shows AI training estimate (random percentage) or declined message

## Design

- Dark green background (`#13281F`)
- Minimal, cinematic, sci-fi aesthetic
- No cards - clean, centered content
- Smooth animations throughout
- Fully responsive
